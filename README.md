# Workshop: Your UI, Inside the Chat Window: Building with Apollo & MCP Apps

To follow along with the workshop, refer to the[Odyssey workshop companion course](https://apollographql.com/tutorials/workshop-mcp-apps-intro).

## Project setup

For the initial state of the project:

1. Run `./install.sh`.

1. Navigate to `dev/flyby` and run `npm run dev`.

1. The web app should open up in the browser. Navigate around.

## Final state

1. The final state of the workshop project is in the `final` branch. Run `git checkout final`.

1. Run `./install.sh`.

1. Run `./start_mcp.sh`.

1. In a new terminal window, navigate to `dev/flyby` and run `npm run dev:mcp`.

1. The MCP Jam Inspector should open up in the browser. Ask questions such as "What space locations can I visit?" and try to submit a review for one location.
